StoreBioinfoGUI

StoreBioinfoGUI is a front end to StoreBioinfo making it straightforward to organize the data in the format required by StoreBioinfo. It also makes sure that the data has the meta data needed for future re-use.


StoreBioinfoGUI only contains two settings and both of these are found in the config folder:
  • local_path:
    • the local path where the projects and datasets are to be stored
  • dataset_types:
    • the supported dataset types
To edit the settings, simply open the files in a text browser and edit the fields.

For further questions see the project home page: http://store-bioinfo-gui.googlecode.com.
